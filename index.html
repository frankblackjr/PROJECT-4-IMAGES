The Image

The how to define an image

An Image is defined like so


Syntax: <img src="#" alt="description> height="??px" width="??px"

(You must always define the width and height)
(CSS is best suited for defining width and height)


The Floating Image

A floating Image can move to the left or right of text as seen below.

<img src="#" alt="description style="float: right; hight:??px; width:??px;">

The image map

An image map allows you to create multiple clickable links insde of your image

<img src="image.png" usemap="#webLang"
<map name="webLang">
<area shape="rect" coords="x1, y1, x2, y2" href=stuff.htm>

Think of the hands of a clock when looking for the x and y coords

<area shape="circle" coords="x, y, diameter href=stuff.htm>

The poly value allows you to map any shape you must how ever give both the x & y coordinates for each point.

<area shape="poly" coords="x,y x2,y2 x3,y3 ect... href=stuff.htm>

Syntax:

<div style="background-image:url('clouds.jpg')"> </div>
